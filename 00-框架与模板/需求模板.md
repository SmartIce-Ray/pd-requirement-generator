# 研发需求模板

> 怎么用：复制本文件到 `品牌/需求/需求名/需求填写.md`，把【】里的内容换成实际填写。
> ①②③ 由需求方填，④ 由研发填。份量 / 成本 / 售价不在本表（研发出成品后按成本定价法定）。

---

## ① 一句话需求

> 提示：哪个品牌、做一道什么、给谁在什么场景吃、为什么现在做或对标什么。

给【品牌】做一道【品类 / 什么产品】，给【目标顾客】在【场景】吃。
为什么现在做 / 对标：【一句话】

## ② 参考与审美（重点）

> 提示：贴参考图，逐图说"借的是哪一点、不要什么"，再说成品最终想长什么样。研发缺审美，这段越具体越好。

参考图（图片放进同目录 `参考图/`，这里列文件名 + 来源）：
- 图1 `参考图/【文件名】`（来源：【小红书 / 抖音…】）— 借：【卖相 / 摆盘 / 配色 / 器皿 / 质感 / 命名 / 包装…】
- 图2 …

不要 / 规避：【别人的翻车点、不符合本品牌调性的】
成品想长什么样：【目标效果，一句话；有目标效果图就放进 `参考图/` 并在此注明】

## ③ 产品要求

> 提示：口味方向 + 关键食材；饮品 / 甜品等按品类补冷热、甜度、质地。

品类：【菜品 / 小吃 / 饮品 / 甜品 / 主食】
口味方向：【咸鲜 / 酸辣 / 清爽解腻 / 复合…】
关键食材 / 必含元素：【…】
品类补充（按需）：【饮品：冷热 · 甜度 · 奶基｜甜品：质地 · 是否冷藏｜主食：主料…】
